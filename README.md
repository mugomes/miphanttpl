# MiPhantTPL
É uma biblioteca PHP para separar o HTML do PHP.

Ele é baseado no [Template](https://github.com/raelgc/template) (veja [CRÉDITOS](CREDITS.txt)), com diversas melhorias. O MiPhantTPL foi desenvolvido por Murilo Gomes e é lançado sob a licença LGPL-2.1-only.

## Requerimento

- PHP 8.3 ou superior
